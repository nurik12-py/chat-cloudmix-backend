## Setup Instructions

1. Navigate to the project root directory
2. Create a `.env` file in the root directory of the project
3. Open the `.env` file and set the following environment variables: MONGODB_URI, OPENAI_API_KEY, JWT_SECRET
4. Save the `.env` file.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
